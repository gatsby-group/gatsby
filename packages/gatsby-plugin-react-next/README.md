# gatsby-plugin-react-next

Use React 16 with your Gatsby v1 site! Gatsby v1 ships with React 15 by default so install this plugin to starting using React 16 in your sites. It includes the two recommended polyfills automatically for ES6 map/set so using adding this is drop-in upgrade to React 16.

## Install

`npm install --save gatsby-plugin-react-next`

## How to use

Simply add the plugin to your `gatsby-config.js`.

```javascript
plugins: [
  `gatsby-plugin-react-next`,
]
```
